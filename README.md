# NoClickSeekBar
禁止点击进度条快进的seekbar

    同时优化了体验：
        增加了滑块触摸区域
        增加了滑块等待动画



Step 1. Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.feiyuu:NoClickSeekBar:Tag'
	}
